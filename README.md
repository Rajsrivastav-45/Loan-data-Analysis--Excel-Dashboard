# 📊 Loan Data Analysis Dashboard (Excel)

## 🔹 Project Overview
This project analyzes loan data using Microsoft Excel dashboard.

## 🔹 Features
- Loan distribution analysis
- Age-wise loan trends
- Gender-based loan insights
- Education vs loan status
- Interactive dashboard using slicers

## 🔹 Tools Used
- Microsoft Excel

## 🔹 Key Insights
- Maximum loans are in "Paid Off" category
- Male customers are higher than female
- Most customers belong to college education group
